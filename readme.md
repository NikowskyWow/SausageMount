## ğŸ® How to Use

### 1. Binding the Mount Key (Crucial)
To use the random mount feature, you must assign a keybind:
1.  Press `ESC` in-game and go to **Key Bindings**.
2.  Scroll down to the **Sausage Mount** category.
3.  Bind a key (e.g., `Z`, `Mouse4`) to **"Cast Random Mount"**.
4.  Press that key to mount up!

### 2. Configuration
You can open the configuration menu to manage your mount list:
* **Minimap Button:** Left-Click the Sausage icon.
* **Right-Click Minimap Button:** Drags the icon to a new position.

### 3. Updates
* Click the **"Check Updates"** button in the config window to get the link for the latest version.
