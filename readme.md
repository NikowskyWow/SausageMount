Sausage Mount (WoW 3.3.5a)
Sausage Mount is a smart random mount summoner for World of Warcraft (3.3.5a / Warmane). It automatically handles Ground vs. Flying mounts based on your zone.

Installation
Copy the SausageMount folder into your WoW directory:
World of Warcraft\Interface\AddOns\

How to Use
Manager: Click the Sausage Icon on your minimap to select which mounts to use (Flying vs Ground).

Keybind: Press ESC -> Key Bindings -> Scroll down to Sausage Mount -> Set your key.

Version: 1.0
